# MoviWebAPP

A web application that allows you to manage a personal movie database with multiple users.

## Features
- Create and manage users
- Search and add movies from the OMDB API to your profile
- Change/Update the movie title
- Delete movies from your profile
- Overview of all movies added by all users

## Installation
1. Clone the repository and install the requirements:
   - pip install -r requirements.txt
2. Create a '.env' file in the project root folder
3. Get your own OMDB API key at: https://www.omdbapi.com/apikey.aspx
 - API = https://www.omdbapi.com/
 - API_KEY = (your API key)
4. Run the app.py
